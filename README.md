# Lands of Teutonia

Web estatica para la partida de DnD **Lands of Teutonia**.

## Netlify

Este repositorio esta preparado para Netlify.

- Build command: dejar vacio.
- Publish directory: `outputs`.

El archivo `netlify.toml` de la raiz ya indica a Netlify que publique `outputs`.

## Desarrollo

La web final esta dentro de `outputs`.

Para cambios rapidos:

1. Edita los archivos de `outputs`.
2. Prueba en local.
3. Sube los cambios al repositorio conectado a Netlify.

Netlify desplegara automaticamente cada cambio cuando el repositorio este conectado.
